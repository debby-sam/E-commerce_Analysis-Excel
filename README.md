# Ecommerce Sales Analysis (Excel)

## Overview
This project analyzes an ecommerce sales dataset using Microsoft Excel to uncover sales trends, profitability drivers, and category performance. The analysis emphasizes visual storytelling and KPI-driven decision support.

## Objectives
- Evaluate sales and profit performance
- Identify category-level contributions
- Analyze year-over-year trends
- Highlight top-performing sub-categories
- Create interactive visual summaries

## Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Aggregation
- KPI Analysis
- Visual Analytics

## Dataset Description
The dataset contains transactional ecommerce data including:

- Orders
- Customers
- Products
- Sales
- Profit
- Discounts
- Categories & Sub-categories
- Geographic information

## KPIs Developed
- Total Sales
- Total Profit
- Profit Margin
- Quantity Sold
- Order Volume
- Year-over-Year Metrics

## Visual Analysis

### Sales & Profit Trends
**Combo Chart**
- Monthly comparison of sales vs profit
- Detection of high and low performing periods

### Profit Contribution
**Waterfall Chart**
- Profit distribution by category
- Identification of key profit drivers

### Sales Distribution
**Pie Chart**
- Relative sales share by category
- Category dominance evaluation

### Geographic Performance
**Map Chart**
- State-level sales analysis
- Regional performance patterns

### Top Performers
**Top 5 Analysis**
- Highest revenue-generating sub-categories
- Performance concentration insights

### Year-Over-Year Analysis
**YOY KPIs**
- Sales growth patterns
- Profitability evolution
- Order volume comparison

## Key Findings
- Technology category drives significant profit
- Sales are unevenly distributed across categories
- Certain months show strong profit efficiency
- Sub-category performance is highly concentrated
